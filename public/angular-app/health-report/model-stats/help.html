Help
Filtering
  • "ALL" is a user filter. Select one of the user names and only events for that user will be shown.
  • "Mouse zoom" is a time filter. It allows you to narrow down a certain time period by using the scroll wheel.
  • "Brush bar" is a time filter. It allows you to narrow down a certain time period by grabbing the edges of the grey rectangle and dragging them.
Functionality
  • "Red Dots" indicate all events that took more than 1h for Model Synch Time and 5h for Model Open Times. Hover over it to see more info.
  • "Blue Line" indicates all events that took less than 1h(Synch) or 5h(Open). Both Line and Dots are filtered by the user.
