Color Coding

There are three basic colors used in a Sheets table. They are there to identify if Sheets have any editing tasks and what they are.

  • Green identifies sheets that user is proposing to create in the model. They don't exist in Revit yet.
  • Orange identifies sheets that user is proposing edits to. Edit might be for example a name change.
  • Red identifies sheets that user is proposing to be deleted. They haven't been deleted in Revit yet.
Filtering

Sheets table can be filtered and sorted in the following manner:

  • "Show 10 entries" is a filter that increases number of rows displayed in a table at one time.
  • "ALL" is a Revit model filter. Select model by name, and only Sheets from that model will be shown.
  • "Previous" & "Next" is a page filter. Navigate to next page of Sheets if there are more available.
